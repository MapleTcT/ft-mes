## Evidence Manifest

Use `deploy/nacos/production-migration/nacos-runtime-patch-evidence.example.json`
as the shape for the real production or rehearsal evidence file. The real file
must stay outside git if it contains operational hostnames, ticket links or
private evidence references.

```bash
make production-nacos-runtime-patch-check
make production-nacos-runtime-patch-ready-check NACOS_RUNTIME_PATCH_EVIDENCE=/secure/path/adp-nacos-runtime-patch-evidence.json
```

The strict-ready check requires rendered config baseline evidence, target Nacos
export evidence, signed diff review, signed runtime patch package evidence,
approved publish/restart window, post-publish Nacos smoke, post-publish platform
smoke and rollback evidence. It rejects `.example`, `template` and `sample`
evidence paths as production READY evidence.
